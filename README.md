# Бот-менеджер для публикации постов ВКонтакте

Один бот, который управляет несколькими группами, генерирует тексты через Agnes AI и прикрепляет готовые фото/видео по ссылкам.

## Команды

- `привет` — проверка работы.
- `пост в "Название" на тему "..." [с фото/видео ссылка] через X минут`
